# 🏠 House Price Prediction ML App

This project is a Machine Learning-based web application that predicts house prices based on user-provided features such as number of bedrooms, bathrooms, and area. The model is built using Linear Regression and deployed using Streamlit for real-time predictions.

---

## 🚀 Features

- Predict house prices based on multiple input features
- Data preprocessing and cleaning
- Feature scaling using StandardScaler
- Model training using Linear Regression
- Performance evaluation using MSE and R² score
- Interactive web interface using Streamlit

---

## 🧠 Machine Learning Workflow

1. Data Collection from CSV file  
2. Data Cleaning (handling missing values)  
3. Feature Selection  
4. Train-Test Split  
5. Feature Scaling  
6. Model Training (Linear Regression)  
7. Model Evaluation (MSE, R²)  
8. Deployment using Streamlit  

---

## 🛠️ Tech Stack

- Python  
- NumPy  
- Scikit-learn  
- Streamlit  

---

## 📊 Model Performance

- Mean Squared Error (MSE): Displayed in the application  
- R² Score: Displayed in the application

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/cyber-country/House-Price-Prediction-ML-App.git
```

2.Navigate to project folder:
```bash
cd House-Price-Prediction-ML-App
```
3.Install dependencies:
```bash
pip install -r requirements.txt
```
4.Run the application:
```bash
streamlit run app.py
```
